About the Selling-App

The Selling-App is a simple iOS application designed to manage item quantities and calculate costs dynamically. Built with UIKit and Swift, it allows users to add and subtract items from their shopping cart while updating the cost in real-time.

Key Features:

  • User can adjust the quantity of an item using a stepper.
  • The total cost of the items is calculated automatically (5000 per item).
  • The total cost is passed to a second view page when the user navigates away.

Technology Stack:

  • Swift Programming Language
  • UIKit Framework for UI Design
  • Storyboard for creating the user interface
  • Basic logic for calculating costs based on item quantity

App Overview

The app has two main pages:

  • Page 1: Users adjust the quantity of an item. The total cost is updated accordingly and displayed on the same page.
  • Page 2: The user can view the final cost, which was passed from Page 1 after navigation.

Code Example

Here's a simplified code snippet of the app:

// ViewController.swift
@IBOutlet weak var value: UILabel!
@IBOutlet weak var qty: UILabel!
@IBOutlet weak var cost: UILabel!

@IBAction func add(_ sender: UIStepper) {
    value.text = sender.value.description
    qty.text = sender.value.description
    let qtyInt = Int(sender.value)
    var output = String(qtyInt * 5000)
    cost.text = output
}

The above code allows the user to adjust the quantity using the stepper and updates the cost in real-time.

When the user clicks a button to navigate to Page 2, the cost value is passed using a segue:

// ViewController.swift
override func prepare(for segue: UIStoryboardSegue, sender: Any?) {
    let sendCost = segue.destination as? page2ViewController
    sendCost?.page1Cost = cost.text
}

This functionality is key to keeping the app simple and easy to use. By passing the total cost to the next page, users can see their final price at any time.
